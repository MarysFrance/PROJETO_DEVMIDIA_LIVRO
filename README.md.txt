# 📚 Mentes In-formadas

**PT-BR**  
Página **estática em HTML** sobre o livro **“Mentes In-formadas”**: capa, descrição, informações do autor e link de compra. Inclui metadados para SEO, responsividade móvel e referência a CSS externo.

**EN**  
A **static HTML** landing page for the book **“Mentes In-formadas”**: cover image, description, author info, and a buy link. Includes SEO meta tags, mobile responsiveness, and an external stylesheet hook.

---

## 📁 Arquivos | Files
- `index.html` — página principal | main page  
- `estilo.css` — (opcional) estilos globais | global styles (optional)  
- `livro.JPG` — imagem da capa | cover image  
- `helio-couto.JPG` — foto do autor | author photo

---

## 🔎 Recursos | Features
- `lang="pt-br"`, `charset="UTF-8"` e `meta viewport` para acessibilidade e mobile
- `meta name="description"` para SEO
- Imagens dedicadas (capa e autor)
- Link de compra (Amazon)

---

## 🚀 Como visualizar | How to view
**Localmente | Locally**  
Abra `index.html` no navegador | Open `index.html` in your browser.

**GitHub Pages (opcional) | optional**  
Ative GitHub Pages na branch `main` usando a raiz `/`. A página ficará pública via a URL do projeto.
